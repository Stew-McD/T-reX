WasteAndMaterialFootprint documentation
=====================================================

The WasteAndMaterialFootprint tool is a python package that allows one to easily and flexibly calculate the waste and material footprint of any product or service inside of life cycle assessment (LCA) databases. Currently is has been tested with ecoinvent 3.8, 3.9, 3.9.1 and 3.10.
